Oblique Anamorphic Image Converter.



About the Oblique Anamorphic Image Converter:

This program is to convert images from one perspective to another, using the oblique anamorphic method. To view the converted image the viewer must tilt the image at a 45 degree angle.

To view the example images below please tilt your device forward by a 45 degree angle.

Please contact erespace@gmail.com , if you would like an image converted using this method and if you have any questions regarding this program.
